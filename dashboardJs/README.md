# Título del Proyecto

Dashbord de ejercicios con JS

## Descripción 🚀

Construido con Html, CSS ,JS se elige una fuente de Google fonts.

### Pre-requisitos 📋

Tener un navegador Web

### Instalación 🔧

Descarga y ejecuta el archivo index.html
